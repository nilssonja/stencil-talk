Types of Custom Elements

  1. Autonomous custom elements 
          
        <my-button></my-button>
  2. Customized built-in elements 
          <button is="my-button"></button>
According to the spec, there are 2 types of custom elements. The first is an Autonomous custom element, which is defined entirely by the author. That is, they do not extend the functionality of existing HTML elements. It is up to the author to define the element's behavior and style from scratch.

The second type of custom element is the customized built-in element. These extend the functionality of an existing HTML element. This part of the spec is still a work in progress and not yet supported by WebKit.

So when people refer to custom elements today, they likely mean autonomous custom elements.
