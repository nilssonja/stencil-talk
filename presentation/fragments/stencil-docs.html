Doc Generation (JSON, Markdown, Custom)

  
    stencil build --docs-readme --docs-json ./docs.json
Lastly, Stencil has documentation generation as part of the build process. With JSDOC comments above your code, Stencil generates JSON describing each of your components, that you can use to build your own docs. Additionally, Stencil can generate markdown files with auto-generated prop and event tables you can view in github or gitlab next to your component code.
